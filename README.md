# MiniAbsorbOverlay

MiniAbsorbOverlay is a World of Warcraft unit-frame enhancement addon that displays absorb shields as an overlay on health bars.

It supports Blizzard's default unit frames and **NDui party frames**.

## Features

- Displays absorb shields as a health-bar overlay
- Supports Blizzard player, target, and focus frames
- Supports Blizzard party and raid frames
- Supports the Personal Resource Display
- Supports NDui party frames
- Uses Blizzard's heal-prediction API to detect clamped absorb shields

## Installation

1. Download or clone this project.
2. Make sure the addon directory is named `MiniAbsorbOverlay`.
3. Place the directory in your World of Warcraft addon folder:

   ```text
   World of Warcraft/_retail_/Interface/AddOns/
   ```

4. Restart the game, or reload the addon list from the character selection screen.

## Supported Game Versions

- Retail
- Classic
- Cataclysm Classic

Refer to the `Interface` values in [MiniAbsorbOverlay.toc](MiniAbsorbOverlay.toc) for the exact supported client versions.
